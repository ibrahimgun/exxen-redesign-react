# Exxen Redesign with React.JS

## Proje Kurulumu
Clone Project:
```
https://github.com/ibrahimgun/exxen-redesign.git

```
```
cd exxen-redesign

```
Install:
```
npm install

```
Start a local server:
```
npm run start

```
